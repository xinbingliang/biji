# 接口

## 网络

### 发起请求

* url  开发者服务器接口地址
* data 请求的参数
* header 设置请求的header、header中不能设置Referer
* method 默认GET 有效值OPTIONS, GET, HEAD, POST, PUT, DELETE, TRACE, CONNECT
* success 收到开发者服务成功返回的回调函数，res = {data: '开发者服务器返回的内容'}
* fail 接口调用失败的回调函数
* complete 接口调用结束的回调函数（调用成功、失败都会执行）
