# shell中函数和数组的使用

## 函数

```bash
#!/bin/bash

function print_welcome(){
	echo "`date`";
}

print_welcome
```

````bash
#!/bin/bash
function sum(){
	if [ $# -eq 2 ]; then
		a1=$1
		a2=$2
		let sum=a1+a2
	else
		echo "参数数量不对"
		break
	fi
	echo "$a1+$a2=$sum"
}

sum 10 101
````

## 数组

```
xin@ubuntu:~/shell$ myarr=(1 2 3 4 4)
xin@ubuntu:~/shell$ echo ${myarr[*]}
1 2 3 4 4
xin@ubuntu:~/shell$ echo ${myarr[1]}
2
```
