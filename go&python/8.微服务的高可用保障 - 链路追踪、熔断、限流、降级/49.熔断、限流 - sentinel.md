# 熔断、限流 - sentinel

## 服务雪崩

![](../images/服务雪崩原因.png)

![](../images/雪崩应对策略.jpg)

熔断：

1. 请求都很慢-很高比例慢
2. 出现大比例请求错误
3. 单位时间内错误数量多

当A出现以上情况，那么A向B请求的服务，主动断开，一段时间后重新发送
