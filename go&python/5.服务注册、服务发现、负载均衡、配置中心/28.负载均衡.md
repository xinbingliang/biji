# 负载均衡

## 负载均衡策略

1. 集中式load balance
2. 进程内load balance
3. 独立进程load balance

## 常用负载均衡算法

1. 轮询法

   资源平均分配

2. 随机法

3. 源地址哈希

   加减服务器会有问题，用一致性哈希实现

4. 加权轮询法

5. 加权随机法

6. 最小连接

   根据已经连接数量，来决定分配
