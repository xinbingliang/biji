## 查找文件夹下`.txt`文件

````python
import os

def findfile(inputdir):
    for parent,dirnames,filenames in os.walk(inputdir):

        for filename in filenames:
            if os.path.splitext(filename)[1] == '.txt':
                print(parent+'\\'+filename)
````
