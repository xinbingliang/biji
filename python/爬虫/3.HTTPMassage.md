# HTTPMassage

## 方法

* `info()`返回`httplib.HTTPMassage`实例


* `httplib.HTTPMassage`
  * `Headers` 头信息
  * `gettype()` 获取它的媒体类型
  * `getheader()/getheaders()` 从打印的头信息中获取到头的字段
  * `Items()/keys()/values()`打印出所有的头出来

## 示例

````python
import urllib

def print_list(list):
    for i in list:
        print(i)

def demo():
````
