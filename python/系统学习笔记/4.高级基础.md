## 模块导入

* `sys.path` 查看模块导入路径
* `sys.path.append("pathto")` 添加自己的模块路径或者在当前项目文件夹下使用`.pth`


## 深浅拷贝

给地址是浅拷贝，给内容是深拷贝

```python
import copy
c = copy.deepcopy(a)#深拷贝

a = [11, 22, 33]
b = [44, 55, 66]
c = [a, b]
e = copy.deepcopy(c) #不是对c的引用而是复制值

c = (a, b)
e = copy.copy(c) #此时e和c是同样的对象
```

 不可变类型直接浅拷贝
