## re模块

### match

* `re.match(正则表达式，将要匹配当字符串)` 返回匹配对象，否则返回None

  ```python
  import re

  pattern ='xin'
  result = re.match(pattern, 'xinbingliang')
  print(result.group())   # 查看匹配出的内容
  ```

## 表示字符

* `.` 匹配任意字符串
* `[]` 匹配`[]`中的任意字符
* `\d` 匹配数字，0-9
* `\D` 匹配非数字，即不是数字
* `\s` 匹配空白，即空格，tab建、`\r`、`\n`、`\t`
* `\S` 匹配非空白
* `\w` 匹配单字符, a-z、A-Z、0-9
* `\W` 匹配非单字符
