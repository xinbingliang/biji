# 面向对象

## 定义

```python
class Cat:
    def talk(self):
        pass

    def eat(self):
        pass
    
cat = Cat();
```

* 即便在类中没有定义属性，也可以直接给对象添加，self作用是传递当前调用对象本身

## 魔术方法

* `__init__(self)`

```python
class Cat:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def talk(self):
        print("名字：%s---年龄：%s" % (self.name, self.age))
    
cat = Cat('辛丙亮', '24');
cat.talk()
```
