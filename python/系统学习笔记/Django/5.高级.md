## 静态文件处理

- 在settings 文件中定义静态内容

```
STATIC_URL = '/static/'
STATICFILES_DIRS = [
    os.path.join(BASE_DIR, 'static'),
]
```

- 在项目根目录下创建static目录，再创建当前应用名称的目录

```
mysite/static/myapp/
```

- 在模板中可以使用硬编码

```
/static/my_app/myexample.jpg
```

- 在模板中可以使用static编码

```
{ % load static from staticfiles %}
<img src="{ % static "my_app/myexample.jpg" %}" alt="My image"/>
```
