# 面向对象

* 仅仅支持封装，不支持继承和多态

## 创建

````go
package main

import "fmt"

type treeNode struct {
	value int
	left, right *treeNode
}

func main() {
	var root treeNode
	fmt.Println(root)

	root = treeNode{value: 3}
	root.left = &treeNode{}
	root.right = &treeNode{5, nil, nil}
	root.right.left = new(treeNode) // new返回的是地址

	nodes := []treeNode{
		{value: 3},
		{},
		{6, nil, &root},
	}
	fmt.Println(nodes)
}
````

## 工厂函数

````go
package main

import "fmt"

type treeNode struct {
	value       int
	left, right *treeNode
}

func createTreeNode(value int) *treeNode {
	return &treeNode{value: value}
}

func main() {
	var root treeNode
	root = treeNode{value: 3}
	root.left = &treeNode{}
	root.right = &treeNode{5, nil, nil}
	root.left.right = createTreeNode(2)

	fmt.Println(root)
}
````

## 变量存放位置

若变量会被其他位置引用例如局部变量取地址后被返回则在堆上分配，若变量不被其他位置引用则在栈上分配

## 方法，接收者值接受和指针接收

```go
package main

import "fmt"

type treeNode struct {
	value       int
	left, right *treeNode
}

func (node treeNode) print() { //此处是值传递，go语言参数传递都是值传递
	fmt.Println(node.value)
}

func (node *treeNode) setValue(value int) {
	node.value = value
}

func createNode(value int) *treeNode {
	return &treeNode{value: value}
}

func main() {
	var root treeNode
	root = treeNode{value: 3}
	root.left = &treeNode{}
	root.right = &treeNode{5, nil, nil}
	root.right.left = new(treeNode)
	root.left.right = createNode(2)

	root.print()
	root.right.left.setValue(4)
	root.right.left.print()

	pRoot := &root
	pRoot.print()
	pRoot.setValue(200)
	pRoot.print()
}
```

## 使用nil指针来调用方法

```go
package main

import "fmt"

type treeNode struct {
	value       int
	left, right *treeNode
}

func (node treeNode) print() { //此处是值传递，go语言参数传递都是值传递
	fmt.Println(node.value)
}

func (node *treeNode) setValue(value int) {
	if node == nil {
		fmt.Println("setting nil!")
		return
	}
	node.value = value
}

func createNode(value int) *treeNode {
	return &treeNode{value: value}
}

func main() {
	var root treeNode
	root = treeNode{value: 3}
	root.left = &treeNode{}
	root.right = &treeNode{5, nil, nil}
	root.right.left = new(treeNode)
	root.left.right = createNode(2)

	root.print()
	root.right.left.setValue(4)
	root.right.left.print()

	pRoot := &root
	pRoot.print()
	pRoot.setValue(200)
	pRoot.print()

	var nRoot *treeNode
	nRoot.setValue(10012)
	nRoot = &root
	nRoot.setValue(301)
	nRoot.print()
}
```

## 树状对象的遍历

````go
package main

import "fmt"

type treeNode struct {
	value       int
	left, right *treeNode
}

func (node treeNode) print() { //此处是值传递，go语言参数传递都是值传递
	fmt.Println(node.value)
}

func (node *treeNode) setValue(value int) {
	if node == nil {
		fmt.Println("setting nil!")
		return
	}
	node.value = value
}

func createNode(value int) *treeNode {
	return &treeNode{value: value}
}

func (node *treeNode) traverse() {
	if node == nil {
		return
	}
	node.left.traverse()
	node.print()
	node.right.traverse()
}

func main() {
	var root treeNode
	root = treeNode{value: 3}
	root.left = &treeNode{}
	root.right = &treeNode{5, nil, nil}
	root.right.left = new(treeNode)
	root.left.right = createNode(2)
	root.right.left.setValue(4)

	root.traverse() // 0 2 3 4 5
}
````

## 接收者怎么选

* 要改变内容必须使用指针接收者
* 结构过大，考虑使用指针接收者
* 一致性：可以是指针接收者，尽量使用指针接收者
