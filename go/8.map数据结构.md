# map

* key-value数据结构，集合
* `var map变量名 map[keytype]valuetype`
* key可以类型
  * key的类型可以是：bool，数字，string，指针，channel还可以为前面几种类型的接口，结构体和数组

## 声明

声明不会分配内存

````go
package main // 打包

import (  // 引入包
	"fmt"
)

func main()  {
	// 声明和注意事项
	var a map[string]string
	fmt.Println(a) //map[]
	//使用map前，需要先make，make的作用就是给map分配数据空间
	a = make(map[string]string, 10)
	a["no1"] = "崔学智"
	fmt.Println(a) // map[no1:崔学智]
}
````

### 增删改查

* 删除

  `````go
  package main // 打包

  import (  // 引入包
  	"fmt"
  )

  func main()  {
  	heros := map[string]string{"heros1": "超人","heros2": "超人",}
  	delete(heros, "heros1")
  	fmt.Println(heros)
  }
  `````

* 查找

  ````go
  package main // 打包

  import (  // 引入包
  	"fmt"
  )

  func main()  {
  	heros := map[string]string{"heros1": "超人","heros2": "超人",}
  	val, ok := heros["heros1"]
  	if ok {
  		fmt.Println(val)
  		fmt.Println(ok)
  		fmt.Println("有值")
  	} else {
  		fmt.Println("没有值")
  	}
  }
  ````
